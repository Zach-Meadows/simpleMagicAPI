# Simple Magic: The Gathering API

this magic api is used to supply modern legal card images to the magic mode in my hangman game. 

Models/Crud Methods
---------
- Cards
  - Create
  - Read
  - Update
  - Delete
- Sets
  - Create
  - Read
  - Update
- Types
  - Create
  - Read
  - Update
  
Features/Uses
---------
- find all modern cards
- find random cards
- find cards by id
- find cards by name
- create new cards
- update cards by id
- delete cards

- find all modern sets
- find sets by id
- find sets by name
- update sets by id
- create new sets

- find all types
- find types by name
- create new types
- update sets
